# CodeAlpha Hangman Game

A simple text-based Hangman Game developed using Python as part of the CodeAlpha Python Programming Internship.

## Features
- Random word selection
- Letter-by-letter guessing
- 6 incorrect attempts allowed
- Displays guessed letters
- Win and lose conditions
- Beginner-friendly console interface

## Technologies Used
- Python
- Random Module

## How to Run
1. Download the project files.
2. Open the project in VS Code or any Python IDE.
3. Run the following command:

```bash
python hangman.py
```
